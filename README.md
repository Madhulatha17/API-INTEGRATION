# API-INTEGRATION

## PROJECT OVERVIEW

This project demonstrates API Integration using HTML, CSS, and JavaScript. The application fetches real-time weather data from the OpenWeatherMap API and dynamically displays it to users.

## FEATURES

- API Integration
- Real-Time Weather Data
- Dynamic Data Loading
- Responsive Design
- Error Handling
- User-Friendly Interface

## TECHNOLOGIES USED

- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API

## HOW TO RUN

1. Download or clone the repository.
2. Get a free API key from OpenWeatherMap.
3. Replace YOUR_API_KEY in script.js.
4. Open index.html in a browser.
5. Enter a city name and view weather data.

## CONCLUSION

The project successfully demonstrates API integration by fetching and displaying real-time weather information from a public API.
